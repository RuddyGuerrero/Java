# Ciclos en JAVA

Los cilcos son sentencias que se repiten.  
En Java existen cuatro tipos de ciclos.  
En todos los ciclos, los pasos se repiten hasta que la condición se cumpla o deje de cumplirse.

# 1. WHILE Y DO…WHILE

## WHILE

### Características

- La condición se verifica primero.
- Si la condición es verdadera, se ejecuta el bloque.
- Si es falsa, no se ejecuta nunca.
- Es un ciclo pre‑probado.

### Sintaxis

```java
while (condition) {
    // statements
}
```

### Ejemplo

```java
class Test {
    public static void main(String args[]) {
        int i = 1, n = 100;

        while (i < n) {
            System.out.println(i);
            i = i * 2;
        }
    }
}
```

## DO…WHILE

### Características

- Primero se ejecuta el bloque.
- Luego se verifica la condición.
- Es un ciclo post‑probado.
- Si la condición es falsa, el bloque se ejecuta al menos una vez.

### Sintaxis

```java
do {
    // statements
} while (condition);
```

### Ejemplo

```java
class Test {
    public static void main(String args[]) {
        int i = 1, n = 100;

        do {
            System.out.println(i);
            i = i * 2;
        } while (i < n);
    }
}
```

# 2. FOR

### Características

- Es un ciclo controlado por contador.
- Es el más usado en Java.
- Tiene tres partes: inicialización, condición y actualización.

### Sintaxis

```java
for (initialization; condition; update) {
    // statements
}
```

### Ejemplo

```java
class Test {
    public static void main(String args[]) {
        for (int i = 1; i <= 10; i++) {
            System.out.println(i);
        }
    }
}
```

# 3. CICLOS ANIDADOS

Un ciclo dentro de otro ciclo se llama ciclo anidado.  
Se pueden anidar cualquier tipo de ciclos.

### Ejemplo 1: for simple

```java
class Test {
    public static void main(String args[]) {
        for (int i = 1; i <= 10; i++) {
            System.out.println(i);
        }
    }
}
```

### Ejemplo 2: ciclos for anidados

```java
class Ejemplo {
    public static void main(String args[]) {
        for (int i = 1; i < 5; i++) {
            for (int j = 1; j < 5; j++) {
                System.out.println(i + "," + j);
            }
        }
    }
}
```
