# Clase — Estructura básica de un programa en Java

## 1. Introducción

Todo programa en Java sigue una **estructura básica**.  
Esta estructura normalmente contiene:

- Importación de bibliotecas
- Declaración de una clase
- Método principal `main`
- Instrucciones del programa

## 2. Importación de bibliotecas

En Java podemos importar bibliotecas usando la palabra **import**.

Ejemplo:

```java
import java.lang.*;
```

La biblioteca `java.lang` contiene clases fundamentales del lenguaje como:

- String
- System
- Math
- Object

⚠️ **Importante:**

Esta biblioteca no es obligatoria importarla, porque Java la importa automáticamente. Incluso si no escribimos `import java.lang.*;`, el programa seguirá funcionando.

---

## 3. Ejemplo de programa

```java
import java.lang.*;

class MyFirst {
    public static void main(String arg[]) {
        System.out.println("Hello World");
    }
}
```

## 4. Explicación del programa

### 4.1 Importación
```java
import java.lang.*;
```
- Importa todas las clases del paquete `java.lang`.

### 4.2 Declaración de la clase
```java
class MyFirst
```
- Declaramos una clase llamada `MyFirst`.  
- Todo el código en Java debe estar dentro de una clase.

⚠️ **Regla importante:** El nombre del archivo debe ser `MyFirst.java`.

### 4.3 Método principal
```java
public static void main(String arg[])
```

- Este es el método principal donde inicia la ejecución.

### 4.4 Mostrar texto en pantalla

```java
System.out.println("Hello World");
```

- Este comando imprime texto en la consola.
- Resultado: `Hello World`

## 5. Compilar el programa

Abrir la consola (CMD) y escribir:

```bash
javac MyFirst.java
```

- Se genera el archivo `MyFirst.class` con el bytecode que ejecutará la JVM.

## 6. Ejecutar el programa

```bash
java MyFirst
```

- Resultado: `Hello World`

## 7. Experimento

- Cambiar el nombre de la clase a `MySecond`.
- Compilar `javac MyFirst.java` y ver que pasa.

## 8. Flujo de ejecución del programa

```
MyFirst.java
      ↓
Compilador (javac MyFirst.java)
      ↓
MyFirst.class
      ↓
JVM (java MyFirst)
      ↓
Se ejecuta el programa
```
