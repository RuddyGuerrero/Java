# Entrada con Scanner y Salida en Java

## 1. Introducción

La clase **Scanner** permite leer distintos tipos de datos desde la entrada estándar (`System.in`). Para mostrar resultados en pantalla se utiliza **System.out.println** o **System.out.print**. También podemos explorar métodos y clases usando `javap`.

## 2. Métodos más comunes de Scanner

| Método | Tipo de dato | Descripción |
|--------|-------------|------------|
| `nextLine()` | `String` | Lee toda la línea hasta un salto de línea. |
| `next()` | `String` | Lee la siguiente palabra (hasta un espacio). |
| `nextInt()` | `int` | Lee un número entero. |
| `nextDouble()` | `double` | Lee un número decimal (double). |
| `nextFloat()` | `float` | Lee un número decimal (float). |
| `nextLong()` | `long` | Lee un número largo (long). |
| `nextBoolean()` | `boolean` | Lee un valor booleano (`true` o `false`). |
| `hasNext()` | `boolean` | Comprueba si hay más entradas (cualquier tipo). |
| `hasNextInt()` | `boolean` | Comprueba si el siguiente token es un entero. |
| `hasNextDouble()` | `boolean` | Comprueba si el siguiente token es un double. |


## 3. Ejemplo de Entrada y Salida

```java
import java.util.Scanner;

public class EntradaSalida{
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        // Entrada de datos
        System.out.print("Ingresa tu nombre: ");
        String nombre = sc.nextLine();

        System.out.print("Ingresa tu edad: ");
        int edad = sc.nextInt();

        // Salida de datos
        System.out.println("Hola " + nombre + ", tienes " + edad + " años.");
        System.out.print("Gracias por usar el programa.\n");
    }
}
```

- `System.out.println()` → imprime y salta a la siguiente línea.
- `System.out.print()` → imprime sin salto de línea.

## 4. Explorar la clase Scanner con `javap`

El comando `javap` permite ver los métodos, constructores y campos de cualquier clase compilada.

```bash
javap java.util.Scanner
```

Esto muestra todas las clases, métodos y constructores públicos de `Scanner`.

## 5. Resumen

- `Scanner` permite leer distintos tipos de datos.
- `System.out.println` y `System.out.print` muestran información en pantalla.
- `hasNext` y variantes permiten validar la entrada.
- `javap` permite explorar métodos y clases de la API de Java.
