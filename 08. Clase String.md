# Trabajar con String en Java

## Qué es un String
String es una clase que representa texto y es **inmutable** (no se puede modificar una vez creado).

## Declaración de Strings más comunes

```java
String str1 = "Java";
String str2 = new String("Java");
```

## Crear Strings desde arreglos de caracteres o bytes

```java
package stringpractice;

public class StringPractice {

    public static void main(String[] args) {

        String str1 = "Java Program";
        System.out.println(str1);

        String str2 = new String("JAVA");
        System.out.println(str2);

        char c[] = {'H','e','l','l','o'};
        //String str3 = new String(c);
        String str3 = new String(c, 1, 3);   // desde índice 1, longitud 3 → "ell"
        System.out.println(str3);

        byte b[] = {65, 66, 67, 68};
        //String str4 = new String(b);
        String str4 = new String(b, 1, 2);   // desde índice 1, longitud 2 → "BC"
        System.out.println(str4);

        String s1 = "Java";              // s1 apunta al String del pool
        String s2 = new String("Java");  // s2 apunta a un objeto nuevo fuera del pool
        String s1 = "Java";              // s3 apuntan al mismo objeto que s1 en el pool
        System.out.println(s1 == s2);    // false (comparación de referencias)
        System.out.println(s1 == s3);    // true (ambos apuntan al mismo literal en el pool)
    }
}
```

## Acceder a un carácter

```java
String s = "Hola";
char c = s.charAt(0);   // devuelve un char
```

## ¿Puedo modificar un carácter?
No. Los Strings son inmutables, no permiten cambiar un carácter directamente.

```java
String s = "Hola";
s[0] = 'M';   // Error
```

### Para modificar un carácter debo crear un nuevo String

```java
String s = "Hola";
String nuevo = "M" + s.substring(1);   // "Mola"
```

### Alternativa si necesito modificar texto

```java
StringBuilder sb = new StringBuilder("Hola");
sb.setCharAt(0, 'M');   // "Mola"
```

# Métodos principales (explicación en una línea + tipo de dato que devuelven)

```java
t.length();              // int → cantidad de caracteres  
t.charAt(0);             // char → carácter en la posición indicada  
t.toUpperCase();         // String → copia en mayúsculas  
t.toLowerCase();         // String → copia en minúsculas  
t.trim();                // String → elimina espacios al inicio y final  
t.contains("Mundo");     // boolean → indica si contiene el texto  
t.indexOf("a");          // int → primera posición donde aparece o -1  
t.lastIndexOf("a");      // int → última posición donde aparece o -1  
t.substring(2);          // String → subcadena desde la posición indicada  
t.substring(2, 5);       // String → subcadena entre inicio y fin-1  
t.replace("o", "0");     // String → reemplaza texto por otro  
t.startsWith("Hola");    // boolean → comprueba si empieza con ese texto  
t.endsWith("Mundo");     // boolean → comprueba si termina con ese texto  
t.split(" ");            // String[] → divide el texto en partes  
t.equals("hola");        // boolean → compara contenido exactamente  
t.equalsIgnoreCase("hola"); // boolean → compara ignorando mayúsculas  
```

# Expresiones regulares (versión muy resumida)

## Qué son

Una expresión regular (regex) es un patrón para buscar o validar texto.

## Cómo se usan en Java

```java
"12345".matches("\\d+");   // forma rápida

Pattern p = Pattern.compile("\\d+");
Matcher m = p.matcher("12345");
m.matches();               // forma completa
```

## Patrones básicos

| Patrón | Significado |
|--------|-------------|
| `\\d` | dígito (0–9) |
| `\\w` | letra o número |
| `[a-z]` | letra minúscula |
| `[A-Z]` | letra mayúscula |
| `.`     | cualquier carácter  |

## Expresiones regulares (operadores)

| Patrón   | Significado                     |
|----------|---------------------------------|
| `+`      | una o más veces                 |
| `*`      | cero o más veces                |
| `?`      | cero o una vez                  |
| `{n}`    | exactamente n repeticiones      |
| `{n,m}`  | entre n y m repeticiones        |
| `^`      | inicio del texto                |
| `$`      | final del texto                 |


## Ejemplos útiles

```java
"1234".matches("\\d+");              // solo números  
"Hola".matches("[A-Za-z]+");         // solo letras  
"Hola".matches("[A-Z].*");           // empieza por mayúscula  
"correo@gmail.com".matches("\\w+@\\w+\\.\\w+");  // email básico 
```
