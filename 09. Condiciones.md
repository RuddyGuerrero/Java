# Condicionales en Java

Las condicionales permiten que un programa tome decisiones basadas en expresiones que se evalúan como verdaderas o falsas.

## 1. Operadores relacionales

Se utilizan para comparar valores y siempre devuelven un booleano.

| Operador | Significado |
|----------|-------------|
| ==       | Igual a |
| !=       | Diferente de |
| >        | Mayor que |
| <        | Menor que |
| >=       | Mayor o igual |
| <=       | Menor o igual |

Ejemplo:

```java
if (edad >= 18) {
    // código
}
else{
    // código
}
```

## 2. Operadores lógicos

Permiten combinar condiciones.

| Operador | Nombre | Ejemplo | Significado |
|----------|--------|---------|-------------|
| &&       | AND    | a > 0 && a < 10 | Ambas condiciones deben ser verdaderas |
| ||       | OR     | a < 0 || a > 100 | Al menos una condición debe ser verdadera |
| !        | NOT    | !activo | Invierte el valor booleano |

## 3. Condicional if, else if y else

Estructura básica:

```java
if (condicion) {
    // código si la condición es verdadera
} else if (otraCondicion) {
    // código si la anterior es falsa y esta es verdadera
} else {
    // código si ninguna condición es verdadera
}
```

Ejemplo:

```java
int nota = 75;

if (nota >= 90) {
    System.out.println("Excelente");
} else if (nota >= 50) {
    System.out.println("Aprobado");
} else {
    System.out.println("Reprobado");
}
```

## 4. Switch-case

Se usa para comparar una variable contra varios valores posibles. 

> Note: Sin `break`, el `switch` continúa ejecutando los siguientes `case` aunque no coincidan. Esto se llama *fall-through*.

Estructura:

```java
switch (variable) {
    case valor1:
        // código
        break;
    case valor2:
        // código
        break;
    default:
        // código si no coincide con ningún caso
        break; // no es necesario
}
```

### Valores permitidos en un `switch` en Java

- `byte`
- `short`
- `int`
- `char`
- `String`
- `enum`
- Wrappers: `Byte`, `Short`, `Integer`, `Character`
- Pattern matching con objetos (Java 17+)

Ejemplo:

```java
int dia = 2;

switch (dia) {
    case 1:
        System.out.println("Lunes");
        break;
    case 2:
        System.out.println("Martes");
        break;
    case 3:
        System.out.println("Miércoles");
        break;
    default:
        System.out.println("Día no válido");
        break;
}
```

## 5. Depuración en NetBeans

Depurar permite ejecutar el programa paso a paso para observar el flujo y los valores de las variables.

### Breakpoints

Un breakpoint detiene la ejecución en una línea concreta.  
Se coloca haciendo clic en el margen izquierdo de la línea.

### Iniciar la depuración

Botón Debug Project o la combinación:
    Ctrl + F5

### Control de ejecución

- Step Over: ejecuta la línea actual y pasa a la siguiente.
- Step Into: entra dentro de un método llamado en esa línea.
- Step Out: sale del método actual.
- Continue: reanuda la ejecución hasta el siguiente breakpoint.

### Ventanas útiles

- Variables: muestra los valores actuales de las variables.
- Watches: permite vigilar expresiones específicas.
- Call Stack: muestra la pila de llamadas.
