# Clase 3 — Instalar y Configurar NetBeans para Java

## 1. Introducción

**NetBeans** es un IDE (Integrated Development Environment) que facilita la programación en Java al ofrecer:
- Editor de código con resaltado de sintaxis
- Autocompletado
- Gestión de proyectos
- Compilación y ejecución integradas

## 2. Requisitos previos

- Tener **Java JDK** instalado en tu sistema.
- Conexión a internet para descargar NetBeans.

## 3. Descarga de NetBeans

1. Ir a la página oficial: [https://netbeans.apache.org](https://netbeans.apache.org)
2. Ejecutar el instalador y seguir las instrucciones.

## 4. Instalación

1. Abrir el instalador descargado.
2. Seleccionar el directorio de instalación.
3. Elegir los componentes necesarios (generalmente se selecciona Java SE).
4. Finalizar la instalación.

## 5. Configuración para Java

1. Abrir NetBeans.
2. Ir a **Tools → Java Platforms**.
3. Asegurarse de que el JDK esté registrado.
4. Crear un **nuevo proyecto Java SE**:
   - File → New Project → Java → Java Application
   - Asignar nombre al proyecto y ubicación
   - Marcar "Create Main Class" si quieres que genere la clase principal automáticamente.

## 6. Probar la instalación

1. Escribir un programa simple:

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello World");
    }
}
```

2. Ejecutar el proyecto con el botón **Run** (o F6).
3. Verificar que aparezca la salida en la consola.

## 7. Resumen

- NetBeans es un IDE que facilita la programación en Java.
- Requiere tener el JDK instalado.
- Permite crear, compilar y ejecutar proyectos de Java de forma integrada.
- Siempre asegurarse de que la plataforma Java esté correctamente configurada.
